# scry

Local-first AI codebase memory engine.

## Quickstart (placeholder)

```
# Build
make test

go build ./cmd/scry

# Index your repo
./scry index

# Search
./scry search "query"

# Ask (extractive evidence-based MVP)
./scry ask "How does indexing work?"

# Status
./scry status
```
